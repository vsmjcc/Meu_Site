# Vantuir Junior — Data Engineering Portfolio

Portfólio profissional de **Vantuir Junior**, Senior Data Engineer com foco em construção de ecossistemas de dados ponta a ponta.

Site publicado em: https://vsmjcc.github.io/Meu_Site/

## Objetivo

Este site foi criado para apresentar minha trajetória, stack técnica e principais cases em Engenharia de Dados, Analytics Engineering, BI, Data Warehouse, governança, observabilidade e automações com IA aplicadas a dados.

A proposta é ir além de um currículo online e funcionar como uma vitrine profissional de arquitetura e projetos de dados.

## Principais temas

- Data Engineering
- Analytics Engineering
- Data Warehouse
- Modelagem dimensional
- ETL/ELT com Python e SQL
- PostgreSQL
- dbt
- Dagster
- Metabase e BI
- Governança de dados
- Observabilidade de pipelines
- AI Agents, LLMs, MCPs, skills e n8n
